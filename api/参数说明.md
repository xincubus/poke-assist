# API 参数说明

## `/api/chat` - 聊天接口（推荐）

这是最简单的接口，适合安卓客户端使用。

### 必需参数

- **message** (string): 用户输入的消息
  - 示例: `"盖欧卡对固拉多使用冲浪"`
  - 示例: `"查询喷火龙的种族值"`

### 可选参数

- **context** (array): 对话历史上下文，用于保持多轮对话
  - 格式: `[{"role": "user", "content": "消息"}, {"role": "assistant", "content": "回复"}]`
  - 大多数情况下不需要，可以忽略

### 请求示例

**最简单的用法（只需要 message）：**
```json
{
  "message": "盖欧卡对固拉多使用冲浪"
}
```

**带对话历史（可选）：**
```json
{
  "message": "那伤害高吗？",
  "context": [
    {"role": "user", "content": "盖欧卡对固拉多使用冲浪"},
    {"role": "assistant", "content": "158-188伤害，43.8%概率一击击倒"}
  ]
}
```

### 响应示例

```json
{
  "success": true,
  "type": "damage_calc",
  "response": "0 SpA Kyogre Surf vs. 0 HP / 0 SpD Groudon: 158-188 (90.2 - 107.4%) -- 43.8% chance to OHKO",
  "data": {
    "damage_range": [158, 188],
    "ko_chance": {...}
  }
}
```

## 关于 Swagger UI 中的 additionalProp1/2/3

这些是 Swagger UI 自动生成的**示例键名**，不是实际参数！

当你在 Swagger UI 中看到：
```json
{
  "message": "string",
  "context": [
    {
      "additionalProp1": "string",
      "additionalProp2": "string",
      "additionalProp3": "string"
    }
  ]
}
```

**实际使用时应该是：**
```json
{
  "message": "盖欧卡对固拉多使用冲浪"
}
```

或者如果需要对话历史：
```json
{
  "message": "盖欧卡对固拉多使用冲浪",
  "context": [
    {"role": "user", "content": "你好"},
    {"role": "assistant", "content": "你好！我是宝可梦助手"}
  ]
}
```

## 安卓客户端建议

### 简单实现（推荐）

只使用 `message` 参数，不需要管理对话历史：

```kotlin
// Kotlin 示例
val request = JSONObject().apply {
    put("message", userInput)
}

// 发送 POST 请求到 /api/chat
```

### 完整实现（可选）

如果需要多轮对话，维护一个对话历史列表：

```kotlin
val conversationHistory = mutableListOf<ChatMessage>()

// 用户发送消息
conversationHistory.add(ChatMessage("user", userInput))

val request = JSONObject().apply {
    put("message", userInput)
    put("context", JSONArray(conversationHistory))
}

// 收到响应后
conversationHistory.add(ChatMessage("assistant", response))
```

## 其他接口

### `/api/query` - 数据查询
只需要一个参数：
```json
{
  "query": "查询喷火龙的种族值"
}
```

### `/api/damage-calc` - 伤害计算
需要三个必需参数，其他都是可选的：
```json
{
  "attacker_name": "盖欧卡",
  "defender_name": "固拉多",
  "move_name": "冲浪"
}
```

完整参数示例见 API 文档：http://localhost:8000/docs
