## API 接口

### 1. `/api/chat` - 智能聊天（推荐）
自动识别意图，最适合安卓客户端使用

**示例请求**:
```json
{
  "message": "盖欧卡对固拉多使用冲浪"
}
```

**示例响应**:
```json
{
  "success": true,
  "type": "damage_calc",
  "response": "0 SpA Kyogre Surf vs. 0 HP / 0 SpD Groudon: 158-188 (90.2 - 107.4%) -- 43.8% chance to OHKO",
  "data": {...}
}
```

### 2. `/api/query` - 数据查询
查询宝可梦、招式、特性等信息

### 3. `/api/damage-calc` - 伤害计算
精确控制所有参数的伤害计算
